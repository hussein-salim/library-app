# Library App

A simple web application to manage a collection of books. Users can add, remove, and update the read status of books in their library. The app also uses local storage to persist the library data across sessions.

## Features

- Add new books with details like title, author, number of pages, and read status.
- Remove books from the library.
- Toggle the read status of books.
- Persist library data using local storage.
- Responsive and visually appealing design.

## Technologies Used

- HTML
- CSS
- JavaScript
